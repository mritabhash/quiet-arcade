# Knight Vigil — prompt pack (v2)

Eight cinematic scene images for the home-page knight band. Each prompt
below is **complete** — copy one paragraph, paste it into your image
model, generate. No assembly required.

## How to generate

1. **Do `guard` first** — it defines her. Then attach the guard image as
   the *character reference* for the other seven (Midjourney `--cref`,
   GPT-4o / Gemini reference image, Leonardo character reference, etc.)
   so her face, hair, and armour never drift.
2. **16:9 landscape** (21:9 if your tool supports it), **≥ 1536 px wide**.
3. The site shows a wide slice cropped from the **bottom half** of each
   image, and phones crop the sides — every prompt already tells the
   model to compose for that, so don't fight it if the sky looks empty.
4. Save as PNG or JPG named exactly `guard`, `walk`, `gaze`, `fight`,
   `bonfire`, `sleep`, `sharpen`, `rest` → put them in
   `quiet-arcade/public/knight/` → commit. (WebP conversion happens on
   my side.)
5. If your tool takes a negative prompt:
   `text, watermark, signature, border, frame, extra limbs, deformed hands, grim, gore, low-res`
